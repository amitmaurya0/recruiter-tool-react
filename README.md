### Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/amitmaurya0/recruiter-tool-react.git
    ```

2. Navigate to the project directory:

    ```bash
    cd recruiter-tool-react
    ```

3. Install dependencies:

    ```bash
    npm install
    ```
4. Start the project:

    ```bash
    npm run dev
    ```
